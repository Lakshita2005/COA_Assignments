## 🧠 Key Formulas

### AMAT:
AMAT = Hit Time + (Miss Rate × Miss Penalty)


---

## ⚙️ I/O Techniques Summary

- **Programmed I/O** → CPU polls device (simple but inefficient)
- **Interrupt-driven I/O** → CPU works until interrupted (efficient)
- **DMA** → Direct transfer between memory and I/O (fastest)
